**GeoGrok Quiz** is a premium, framework-free web application designed to challenge and expand your global IQ. Test your knowledge across four distinct categories with a completely dynamic, responsive, and beautifully animated interface.

🖥️ **[Live Demo]https://geole.netlify.app/

---

## ✨ Features

* **100+ Curated Questions:** Balanced across Geography, History, Politics, and International Affairs.
* **Complete Randomization:** Advanced shuffling algorithm ensures no repeated questions and randomized answer choice order every playthrough.
* **Premium Glassmorphism UI:** A modern, high-fidelity dark/light mode interface utilizing native CSS backdrop filters, smooth scaling transitions, and dynamic linear gradients.
* **Web Audio API Sound FX:** Custom synthesized audio frequencies generated entirely through code for correct and incorrect answers—no external audio files required.
* **Interactive Mechanics:** Features a custom circular countdown timer (15 seconds per question), real-time progress bars, and instantaneous feedback highlighting.
* **Persistent Top 5 Leaderboard:** Locally hosted scoring metrics saved via browser `localStorage` with XSS prevention utilities.

---

## 🛠️ Built With

This project is built purely with web standards—**no frameworks, external dependencies, libraries, or hosted APIs required**.

* **HTML5** – Semantic element organization.
* **CSS3** – Custom properties, Flexbox/Grid layouts, custom SVG dash animations, and fluid media queries.
* **Vanilla JavaScript (ES6+)** – Central runtime state architecture, Web Audio API synthesis, and LocalStorage data mapping.
